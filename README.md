## Project Overview

This project is a **Smart Shopping Web Application** that helps users find the best products based on their preferences.

Users can:

* Select category using numbers
* Filter products by price and rating
* View recommended products
* See "Customers Also Bought" suggestions

---

##  Features

*  Product Recommendation System
*  Number-based Category Selection
*  Price Filtering
*  Rating Filtering
*  Product Cards with Images
*  Smart Bundle Suggestions
*  Price Distribution Graph

---

## Technologies Used

* Python
* Streamlit
* Pandas
* Matplotlib

---

##  User Flow

1. Open the app
2. Select category number
3. Adjust price and rating
4. View recommended products
5. Explore related products

---

##  Output

* Product recommendations displayed in card format
* Graph showing price distribution
